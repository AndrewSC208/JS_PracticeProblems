JavaScript Data Types Conversion

Number() converts to a number, String() converts to a String, Boolean() converts to a Boolean.

JavaScript Data Types

JavaScript there are 5 different data types that can contain values:

  • string
  • number
  • boolean
  • object
  • function

There are 3 types of objects:

  • Object
  • Date
  • Array

And 2 data types that cannot contain values:

  • null
  • undefined

the typeof Operator

You can use the typeof operator to find the data type of a JavaScript variable

The constructor Property

The constructor property returns the constructor function for all JavaScript variables.

The global methods Numbers() can convert strings to numbers. String containing numbers (like "3.14") convert to numbers ( like 3.14) Empty strings convert to 0. Anything else converts to NaN (Not a number)
